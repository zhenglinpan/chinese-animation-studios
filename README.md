# 中国动画公司分布地图

一个交互式的中国动画工作室地图，直观展示全国各地的动画制作公司分布情况。

[地图链接](https://zhenglinpan.github.io/chinese-animation-studios/)

## 📖 项目介绍

本项目旨在为动画行业从业者和爱好者提供一个便捷的工具，帮助大家：

- 🗺️ **一目了然**：在地图上直观查看各省份动画工作室的分布和数量
- 🔍 **快速搜索**：通过搜索功能快速找到感兴趣的动画工作室
- 📊 **了解行业**：掌握中国动画产业的地域分布和发展现状
- 🤝 **共建共享**：欢迎所有人贡献和完善工作室信息

### 如何使用

1. **浏览地图**：使用鼠标拖拽、滚轮缩放浏览中国地图
2. **查看工作室**：点击任意省份，右侧会展示该省份的所有工作室列表
3. **搜索工作室**：在顶部搜索框输入工作室名称，快速定位
4. **贡献数据**：点击右下角编辑图标，提交新的工作室信息
